# List secrets

## Basic usage

List vault secrets in JSON format:

```sh copy
dcli list
```

Output:

```sh copy
{
  "credentials": [
    {
      "id": "{92C00759-89E0-43E8-8F9F-7C59D8604D7F}",
      "title": "google.com",
      "url": "https://www.google.com",
      "email": "john.doe@example.com"
    },
    {
      "id": "{7B73F075-1634-4802-A47F-017C52A7A328}",
      "title": "apple.com",
      "url": "https://www.apple.com",
      "email": "john.doe@example.com"
    },
  ],
  "notes": [
    {
      "id": "{A859CEA4-B285-4C35-8701-561EC8A22978}",
      "title": "MySQL DB password"
    },
    {
      "id": "{1C268FFC-A637-4E84-94BB-E588D300E80A}",
      "title": "GitHub SSH Key"
    },
    {
      "id": "{FE4DDA6F-A8FF-4BF6-9A92-00832D761091}",
      "title": "GitLab SSH Key"
    }
  ]
}
```

## With filter

List vault secrets with a search filter applied:

Example 1

```sh copy
dcli list email=john.doe@example.com
```

Output:

```sh copy
{
  "credentials": [
    {
      "id": "{92C00759-89E0-43E8-8F9F-7C59D8604D7F}",
      "title": "google.com",
      "url": "https://www.google.com",
      "email": "john.doe@example.com"
    },
    {
      "id": "{7B73F075-1634-4802-A47F-017C52A7A328}",
      "title": "apple.com",
      "url": "https://www.apple.com",
      "email": "john.doe@example.com"
    },
  ],
  "notes": []
}
```

Example 2

```sh copy
dcli list title="SSH Key"
```

Output:

```sh copy
{
  "credentials": [],
  "notes": [
    {
      "id": "{1C268FFC-A637-4E84-94BB-E588D300E80A}",
      "title": "GitHub SSH Key"
    },
    {
      "id": "{FE4DDA6F-A8FF-4BF6-9A92-00832D761091}",
      "title": "GitLab SSH Key"
    }
  ]
}
```

Example 3

```sh copy
dcli list title=Github
```

Output:

```sh copy
{
  "credentials": [
    {
      "id": "{A307EF37-729A-41FC-A477-D0265331D94D}",

      "id": "{1C268FFC-A637-4E84-94BB-E588D300E80A}",
```
